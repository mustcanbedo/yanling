# 言灵 Yanling

> 小红书评论区 AI 助手 —— 用 AI 回评论，点燃言灵力

## 关于这个仓库

本仓库用于发布言灵 Yanling 扩展的 [Release 版本](https://github.com/mustcanbedo/yanling/releases) 以及使用 [GitHub Issues](https://github.com/mustcanbedo/yanling/issues) 收集和跟进用户反馈。

言灵 Yanling 并非开源软件，这个仓库**不包含**言灵扩展的源代码。

[**点此安装言灵**](https://chrome.google.com/webstore) 或查看产品文档。
